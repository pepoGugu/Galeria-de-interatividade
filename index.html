Processo

Do rascunho ao post

Um fluxo simples para transformar ideias soltas em conteudo publicavel.

1. Defina a pergunta

Todo layout precisa responder uma duvida: explicar, comparar, convencer, resumir ou conduzir.

2. Gere tres formatos

Crie pelo menos tres estruturas diferentes antes de escolher: uma linear, uma modular e uma visual.

3. Teste no mobile

Se o conteudo perde hierarquia em tela pequena, o layout ainda nao esta pronto.

4. Publique e revise

Suba no GitHub Pages, compartilhe o link e registre o que funcionou para a proxima rodada.
